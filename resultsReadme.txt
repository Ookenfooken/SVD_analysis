patient:
0 --> control
1 --> patient

subjectID: 
This doesn't correspond to the actual patient ID, because matlab is really bad with strings. Numbers are in alphabetical order and actual IDs will have to be reassigned later

target.correct (especially important for antisaccade task):
0 --> incorrect target selection
1 --> correct target selection
2 --> correct target selection, but initial saccade in opposite direction

latency: 
initial saccade latency for pro and anti saccade test

cumulative:
cumulative saccade sum

number:
total saccade number

accuracy:
2D distance between final saccade landing position and target location

velocity:
mean saccade velocity
