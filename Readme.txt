The code to analyse the 5 eye movement battery tests was developed by Jolande Fooken in 2018
For questions contact jolande.fooken@rwth-aachen.de

To visually view the data run viewEyeData.m; 
you will be prompted to select the population and have to select the task you want to look at 

To run the automatic analysis run analyzeEyeBattery.m;
you will be prompted to select the population and have to select the task you want analyze
to get a summary of the results in a csv or mat file you need to run
readoutProSaccadeTest.m or
readoutAntiSaccadeTest.m
